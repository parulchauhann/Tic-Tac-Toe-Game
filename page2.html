Key Points
  1. The game is played on a grid that's 3 squares by 3 squares.
  2. Suppose you are X (cross) and your friend is 0 (nought).
  3. Take turns to put mark in empty squares.
  4. First player to get 3 of marks in a row is the winner.
  5. Game is over if no player has 3 marks in a row or the 9 squares grid is full.
